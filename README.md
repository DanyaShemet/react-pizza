# Пиццерия на Реакте
```
Пет проект для практики в React, Redux. Используется json-server
```

## Что бы запустить

### `yarn json`

Запускает тестовый сервер (json-server) на порту 3001

### `yarn start`

Запуск приложения. Без тестового сервера отображения пиц не будет
